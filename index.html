About

From the bench to the backend.

The same rigor that once went into lab protocols now goes into database schemas and API contracts.

I started my career in pharmaceutical chemistry, earning a BSc from Benha University and later a postgraduate diploma, while working as a Medical Laboratory Specialist and in medical representation. Years spent on precise measurements, controlled variables, and documentation that has to be exactly right taught me habits that turned out to translate directly into software engineering.

Today I'm a software engineering student in the Digilians Program's Backend/Full‑Stack Development Track, building production‑style systems on the MERN stack. My graduation work includes a Modular Monolith system built with Clean Architecture, complete with full software architecture and security documentation — the same instinct for documenting everything, just aimed at code instead of chemistry.

Outside of engineering, I'm an active karate practitioner and referee with the Egyptian Karate Federation. Discipline, repetition, and getting the fundamentals right before moving on are as central to how I write software as they are to how I compete.
