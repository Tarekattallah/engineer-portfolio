## ✨ Key Features

- **Dark & Immersive UI** — A sophisticated, code-first aesthetic that highlights technical proficiency.
- **Terminal Typing Effect** — A dynamic intro that immediately establishes the engineering context.
- **Martial Arts Proficiency Scale** — Skills are visualized using a "Belt Ranking" system, inspired by my background in Karate.
- **Fully Responsive** — Seamless experience across all devices, from 4K monitors to mobile screens.
- **Scroll-Reveal Animations** — Smooth, performant animations triggered as you navigate the page.
- **Interactive Contact Form** — Front-end validation ready to be connected to a backend service (EmailJS, Nodemailer, etc.).

---

## 🛠️ Built With

This project is 100% static and requires zero build tools or package managers.

- **HTML5** — Semantic, accessible markup.
- **CSS3** — Custom properties (variables), modern grid/flexbox layouts, and subtle gradient overlays.
- **Vanilla JavaScript** — Intersection Observer API for scroll reveals, DOM manipulation for the typing effect, and event handling for the mobile navigation.
